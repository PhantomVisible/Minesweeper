# Minesweeper Game

A web-based Minesweeper game built with HTML, CSS, and JavaScript. This project replicates the classic Minesweeper experience with modern UI improvements and interactive features.

# Features

- Three difficulty levels: Easy, Medium, Hard

- Dynamic board resizing based on difficulty

- Flag mode to mark suspected mines

- Game Over overlay with sound effect when a mine is clicked

- Restart button to quickly start a new game

- Responsive and interactive UI/UX improvements:

    - Hover effects for tiles

  - Adaptive tile sizes for easier gameplay

  - Animated mine reveal

  - Modern font and clean layout

# Installation / Usage

1. Clone the repository or download the files:

```bash
git clone <your-repo-url>
```

2. Open index.html in a web browser (Chrome/Edge/Firefox recommended).

3. Start playing:

   - Select difficulty from the dropdown

   - Click tiles to reveal numbers or mines

   - Use the Flag button below the board to mark suspected mines

# How to Play

1. Choose your difficulty level.

2. Click tiles to reveal numbers or empty spaces.

3. Use the Flag button to mark tiles you suspect contain mines.

4. Avoid clicking mines — if you click one, the game ends with a “BOOM! Game Over” overlay.

5. Clear all non-mine tiles to win the game.
